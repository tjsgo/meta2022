## 라면짱짱걸 홈페이지 프로젝트

### 🍜 프로젝트 제작 계기 🍜 <br/>
✔ 1인가구 증가에 따른 묶음라면 구매 부담스러움 해소 <br/>
✔ 신제품 묶음라면 판매시 구매에 대한 거부감 해소 <br/>
✔ 편의점에서만 구매가 가능한 PB상품 등 물리적 위치의 문제점 해소 <br/>

### 🍜 홈페이지 특징 🍜 <br/>
✔ 다양한 식품사에서 판매하는 라면을 한번에 쇼핑가능 <br/>
✔ 섞어먹는 조합의 라면을 1봉지 단위로 구매가능 <br/>
✔ 라면 신상품 등 최신정보 확인가능 <br/>
